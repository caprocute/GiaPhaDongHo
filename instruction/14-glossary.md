# Glossary thuật ngữ GiaPhaHub

| Thuật ngữ VN | Mã / entity | Ý nghĩa |
|--------------|-------------|---------|
| Gia phả / cây | `FamilyTree` | Một dòng họ / chi (multi-tree) |
| Hồ sơ người | `Person` | Thành viên trong cây |
| Hôn phối | `FamilyUnion` + `UnionMember` | Cặp vợ chồng / phối ngẫu |
| Con của cặp | `UnionChild` | Liên kết con với union |
| Đời | `generation` | Thứ tự đời (cache) |
| Mã hiệu | `Person.code` | VD `A7`, `A7-sp1` |
| Phả ký / tộc ước / hương hỏa | `Chapter.kind` | Văn bản chương |
| Ngày giỗ | `DeathAnniversary` | Giỗ âm lịch gắn người đã mất |
| Tự khai | `ChangeRequest` | Diff chờ tộc trưởng duyệt |
| Công đức | `DonationCampaign` / contribution | Quỹ / bảng vàng |
